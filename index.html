What does GameCheetah do?

GameCheetah is a AS3 SWC library created solely for the purpose of optimizing workflow in the creation small projects using FlashDevelop.

Designed to be light and flexible, GameCheetah is terrific for prototyping many types of 2D bitmap-based games! Some of its many useful features are:

Easy Graphic Management

The Engine provides a simple Flash UI (using MinimalComps) in the Developer SWC version which provides a clean centralized way to access and manage your sprite sheets, collision objects, as well as live previewing of animations!

Pixel-Perfect Collision Detection

Implementing collision detection has always been a struggle in Flash. Perfect collision detection comes at the cost of performance. GameCheetah offers a simple quadtree search implementation of pixel-perfect collision detection (with rotation) using collision-as3 library.

A Standard Design Pattern!

What really speeds up workflow in game development is utilizing template classes to add game logic. The Entity class in GameCheetah provides basic handlers such as onCollision, onMove, onUpdate, onActivate, onMouseDown, etc.... You can always inherit from Entity derived classes to customize your own handlers!

Another advantage is that this creates really modular classes that can be shared with the community as well!

Always free and Open Source

What's great about many AS3 Engines like Flashpunk, is that you can jump right into the source code to figure out what's going on! Which is why GameCheetah is completely free and open sourced so check it out on Github here!
