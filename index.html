Rawr! Try building your next flash game with GameCheetah!

  • Overview

    GameCheetah is intended to be an AS3 library used for fast, flexible hacking of flash games when using FlashDevelop or FlashBuilder. Designed to be light and flexible, GameCheetah is great for prototyping almost any type of 2D bitmap-based game with every feature carefully thought out to bring value to the workflow of flash game development.

  • Simple User Interface

    The Developer SWC provides a friendly flash interface for managing sprite-sheets, modifying collision bitmap masks, tweaking or previewing sprite-sheet animation, in-game performance testing, and much more!

  • Collision Detection

    Implementing fast, precise collision detection has always been a bit of a struggle in Flash. GameCheetah comes with pixel-perfect, rectangle or even point collision detection with rotation and scaling! Implementing a quadtree search with filtering for collision searching, there's enough optimization to not worry about performance!

  • Performance

    GameCheetah is designed with performance in mind. Using bitmap bliting with render culling optimizations and loop-based event-listening, the demos show a remarkable performance for 1600 game objects with no significant lag to game performance! Additionally, the library offers live performance metrics for monitoring performance.

  • Templates

    What can really speed game prototyping is using prebuilt template classes to add game logic. The Entity class in GameCheetah provides most common event handlers. You can always inherit from Entity derived classes to customize your own handlers! Using templates also makes highly modular classes that can be used across projects.

    Check out the demos to see how to build a variety of games using a few simple handlers!

  • Open Source

    What's great about many AS3 Engines like Flashpunk, is that you can jump right into the source code to figure out what's going on! Which is why GameCheetah is completely free and open sourced so please check it out on Github!
